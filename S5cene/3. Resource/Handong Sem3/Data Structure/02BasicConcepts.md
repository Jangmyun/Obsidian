# 데이터 구조
## SW 개발 단계
1. #### Requirement analysis (요구사항 분석)
2. #### Design (설계)
3. #### Coding (구현)
4. #### Verification (검증)

## Algorithms
#### 어떤 문제를 해결하기 위한 일련의 절차 또는 방법
+ ##### output을 생성
+ ##### 각 수행 단계는 명확하여야 함
+ ##### 각 수행단계는 실제 실현가능한 내용이어야 함
+ ##### 유한단계의 수행 후에 반드시 종료해야 함
